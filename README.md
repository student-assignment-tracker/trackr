# trackr
student assignment tracker
The Student Assignment Development and Tracking software will help instructors create,
manage, and monitor assignments and student submissions. The will allow instructors to design
assignments with deadlines requirements and multiple response types(Multiple choice/Short
Response). The system will record important information such as submission time, assignment
status, and grading results, allowing instructors to efficiently track student progress and
performance. The software can also be used to test program files based on expected output
from a constant input. Centralizing the assignment creation submission and analysis will help
Instructors and Students navigate assignments more effectively.
The project deliverable will be a site hosted on github connected to a backend DB (likely
SupaBase) to contain student and assignment information. The software will allow professors to
create and customize assignments as well as automatically grading assignments (both multiple
choice and code graded automatically) and viewing the data in a digestible way to give insights
into student performance. The database will contain information such as deadlines, grades, and
previous submissions, and our python backend will be used to organize the data to be displayed
to the instructors. On the student end it will have a calendar of your upcoming assignments and
allow you to submit/do the assignments. Students will be assigned to professors based on
classes
